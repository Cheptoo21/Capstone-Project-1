## 💻 Getting Started <a name="getting-started"></a>



To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:


Example command:

```sh
1.use a browser
2.use cable internet
```
 

### Setup

Clone this repository to your desired folder:


Example commands:

```sh
  
  git clone git@github.com:Cheptoo21/Capstone-Project-1.git
  cd Capstone-Project-1



```


### Install

Install this project with:


Example command:

```sh
  npm install
```


### Usage

To run the project, execute the following command:


Example command:

```sh
- Open the index with live server.
- Or if you don't have a live server extension open the index in the browser by clicking on the file .
```


### Run tests

To run tests for the linter, run the following command:


Example command:

```sh
npx hint .
npx eslint .
```
